## 💻 Project Structure

This portfolio is built on a full-stack architecture utilizing Vite, React, and an integrated Express server:

```text
client/                   # React SPA frontend
├── components/ui/        # Radix UI + Tailwind components
├── pages/                # Route components (e.g., Index.tsx, NotFound.tsx)
├── App.tsx               # App entry point & Routing
└── global.css            # Global styles and Tailwind theming

server/                   # Express API backend
├── index.ts              # Main server configuration
└── routes/               # API endpoint handlers

shared/                   # Shared types and utilities (Client & Server)
└── api.ts                
```

## 🏃‍♂️ Getting Started

To run this portfolio locally on your machine, follow these steps:

### Prerequisites
- Node.js (v18 or higher recommended)
- pnpm (preferred package manager)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/luan-portfolio.git
   cd luan-portfolio
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Start the development server (runs both client and backend on port 8080):
   ```bash
   pnpm dev
   ```

4. Visit `http://localhost:8080` in your browser.
